# Exercícios de Vetores em JavaScript

## 📚 Sobre a atividade

Esta atividade foi desenvolvida para praticar conceitos básicos de lógica de programação utilizando JavaScript.

Os exercícios trabalham principalmente com:

- Vetores (Arrays)
- Estruturas de repetição (`for`)
- Condições (`if`)
- Entrada de dados com `prompt()`
- Saída de dados com `console.log()` e `alert()`

- JavaScript
- HTML5
- Visual Studio Code
- Live Server

# 📋 Exercícios desenvolvidos

## ✅ Exercício 1 — Soma dos números pares

### Objetivo
Ler 10 números inteiros, armazenar em um vetor e calcular a soma apenas dos números pares.

### Lógica utilizada
1. Criar um vetor vazio
2. Pedir 10 números ao usuário
3. Guardar os números no vetor
4. Percorrer o vetor novamente
5. Verificar quais números são pares
6. Somar os valores pares
7. Mostrar o resultado

### Conceitos utilizados
- Arrays
- `for`
- `if`
- Operador `%`

---

## ✅ Exercício 2 — Números aleatórios

### Objetivo
Gerar automaticamente 7 números aleatórios e armazenar em um vetor.

### Lógica utilizada
1. Criar um vetor vazio
2. Gerar números aleatórios usando `Math.random()`
3. Armazenar os números no vetor
4. Mostrar os números gerados

### Conceitos utilizados
- Arrays
- `for`
- `Math.random()`
- `Math.floor()`

---

## ✅ Exercício 3 — Contagem de números repetidos

### Objetivo
Solicitar 10 números ao usuário e verificar quantas vezes um número específico aparece no vetor.

### Lógica utilizada
1. Criar um vetor vazio
2. Ler 10 números
3. Pedir um número para pesquisa
4. Percorrer o vetor
5. Contar quantas vezes o número aparece
6. Mostrar o resultado

### Conceitos utilizados
- Arrays
- `for`
- `if`
- Contadores

---

## ✅ Exercício 4 — Remover valores duplicados

### Objetivo
Criar um novo vetor sem números repetidos.

### Lógica utilizada
1. Ler 10 números
2. Criar um segundo vetor vazio
3. Verificar se o número já existe no novo vetor
4. Se não existir, adicionar
5. Mostrar os dois vetores

### Conceitos utilizados
- Arrays
- `for`
- Vetores auxiliares
- Comparações
